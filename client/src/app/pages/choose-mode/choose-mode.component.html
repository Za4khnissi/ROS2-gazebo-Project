Aide et Instructions

Comment utiliser cette interface

Cette interface permet de sélectionner un mode pour interagir avec le robot ou le système. Voici les étapes :

  • Mode Robot Physique : Cliquez sur le bouton "Physical Robot" pour contrôler directement le robot physique.
  • Mode Simulation : Cliquez sur le bouton "Simulation" pour travailler dans un environnement simulé.
  • Base de Données : Cliquez sur le bouton "Database" pour consulter les historiques de missions.

Fonctionnalités principales

  • Identifier le robot : Permet de vérifier l’identité du robot sélectionné (par un son).
  • Démarrer une mission : Lance une mission autonome pour le robot. Vérifiez que l’environnement est sécurisé.
  • Arrêter une mission : Interrompt immédiatement la mission en cours. Une confirmation sera demandée.
  • Retour à la base : Demande au robot de revenir à sa position initiale.
  • Gestion de la batterie : Le robot retourne automatiquement à la base si la batterie descend en dessous de 30 %.

Conseils pratiques

  • Assurez-vous que le robot est dans un environnement dégagé avant de commencer une mission.
  • Pour la simulation, configurez correctement les paramètres (mode de conduite, trajectoires).
  • Consultez les logs régulièrement pour surveiller l’état des missions.
